## ğŸ“‹ Guidelines

### Adding a New Project

1. Create a new directory with a descriptive name (use kebab-case)
2. Add a README.md in the project directory
3. Keep all project files within its directory
4. Update this main README with project info

### Project Directory Naming

- Use lowercase with hyphens: `my-awesome-project`
- Be descriptive but concise
- Include tech stack if relevant: `react-todo-app`

### Project README Template

Each project should have its own README with:
- Project name and description
- Installation instructions
- Usage/Running instructions
- Dependencies
- TODO/Progress tracking
- Any relevant notes
